# MunicipalityApp

## Overview

MunicipalityApp is a Windows Forms application designed for reporting issues to a municipal authority. Users can submit issues by filling out a form with details such as location, category, and description, and attach media if necessary. The application includes a progress bar to indicate the completion status of the report.

## Prerequisites

- **.NET Framework**: Ensure that you have the .NET Framework installed on your machine. The project targets .NET Framework 4.8.
- **Visual Studio**: It is recommended to use Visual Studio 2019 or later for running and modifying the project. 

## Getting Started

To get started with the MunicipalityApp, follow these steps:

### 1. Clone the Repository

Clone the repository to your local machine using Git:

```bash
git clone https://github.com/FortyMads/MunicipalityApp.git
2. Open the Project
Open Visual Studio.
Go to File > Open > Project/Solution.
Navigate to the directory where you cloned the repository and select the MunicipalityApp.sln file.
3. Restore NuGet Packages
In Visual Studio, right-click on the solution in the Solution Explorer.
Select Restore NuGet Packages. This will ensure that all necessary packages are installed.
4. Build the Solution
Go to Build > Build Solution or press Ctrl + Shift + B to compile the project.
5. Run the Application
Set the MunicipalityApp project as the startup project.
Click on the Start button or press F5 to run the application.
The main form will open, and you can interact with the application as intended.
Features
Issue Reporting: Users can report issues by entering details and attaching media.
Progress Tracking: A progress bar updates based on the completion of required fields.
Validation: Input fields are validated to ensure all necessary information is provided.
Usage
Fill in the Location: Enter the location of the issue in the text box.
Select a Category: Choose a category from the drop-down list.
Describe the Issue: Provide a detailed description of the issue in the rich text box.
Attach Media: Click the "Attach Media" button to select and attach an image or other file.
Submit: Click the "Submit" button to submit the issue. The application will validate the inputs and save the issue if all fields are correctly filled.
Error Handling
The application includes error handling for various operations such as file attachment and form submission. Error messages will be displayed in message boxes if any issues occur.
